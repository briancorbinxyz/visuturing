The Diagram Simulator

 

 

The Diagram Simulator in VisuTuring allows you to see in a Transition Diagram how a Turing Machine makes transitions between states given an input word supplied by you. A red dot travels across the current transition and lights up the state as it enters.

Here's how to get started:

  1. Either click on the Simulator tab of your open Turing Machine or go to the Simulate menu and select Transition Diagram.
  2. The Diagram Simulator will be displayed.

  3. The labelled areas function as follows:

    A. Input Word Box - This is where you type your input for the tape. (Press INSERT to insert the blank character)
    B. Play Button - Select to start the simulation with your input word or resume a paused simulation.
    C. Pause Button - Select to pause a running simulation.
    D. Stop Button - Select to stop a running simulation
    E. Speed Buttons - Increase or decrease the speed of the simulation.
    F. Input / Output - Shows the output produced on a given input when a simulation is completed. There is no output if it doesn't reach the halting state.
    G. Command Line - Shows the computation formally as a set of configurations.
    H. Behaviour Panel - (Only appears for non-deterministic machines) Use this to choose how the simulator behaves when it comes across states that can make multiple transitions when it reads a given symbol.
    I. Simulation Area - This is the area where the simulation is displayed.

  4. Type your input word into the input box.
  5. Press play to see the simulator simulate the computation on your word.
  6. You can use Pause or Stop at any time during the computation.

NB: For non-deterministic machines use the Behaviour Panel to choose how the simulator behaves when it comes across states that can make multiple transitions when it reads a given symbol. It can choose the first corresponding transition in your transition table, the last, choose one at random or ask you to decide at the time.

TIP: Try opening two different machines and running simulations at the same time!
