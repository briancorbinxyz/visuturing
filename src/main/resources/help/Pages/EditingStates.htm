Editing the States

 

 

Changing and adding to the states of your Turing Machine can be done from the Edit States Panel:

  1. If it isn't already open, go to the Edit Menu and select States...
  2. The Edit States Dialog will be displayed.

  3. Type in the name of a state you want to add and press Add - it will then be added to the list of existing states.
  4. To remove a state click on it and press Remove.
  5. To remove all states except the starting state s and halting state h press Remove All.
  6. If you are finished, press Okay.
  7. You can cancel changes at any time by pressing the Cancel button.

NB: VisuTuring will stop you from removing any states involved in transitions. You must remove the transitions first.
