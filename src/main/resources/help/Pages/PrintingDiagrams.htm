Printing Diagrams

 

 

To print a diagram:

  1. Open your Turing Machine.
  2. Go to the File menu and select Print...

  3. The default printing style is to leave the document size unchanged. If the whole diagram does not fit on the page (as denoted by the red box) select Fit To Page or select custom and input a size for the diagram in pixels.
  4. You can also choose whether to print the diagram portrait or landscape.
  5. Press Okay.
  6. Choose your printer-specific options from the dialog displayed and select Print.
  7. Your diagram should be printed.
