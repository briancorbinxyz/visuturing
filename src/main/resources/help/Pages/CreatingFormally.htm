Creating Turing Machines Formally

 

 

VisuTuring makes it simple to create Turing Machines by manually adding states, symbols for the Turing Machine alphabet and transitions.

There are several ways to begin this process:

From the Startup Wizard:

  1. Choose Create.

From the main Program:

Either:
  • Go to the File Menu and click New

or:

  • Click the New button on the toolbar

then...

  • The first panel displayed will be the Edit Description Panel, where you can choose a name and description for your new Turing Machine.
  • Use the Back and Next Buttons to switch between panels where you can edit the name and description, edit states, edit the alphabet or edit the transition table and press Okay when you have finished.
  • You can cancel this process at any time by pressing the Cancel button.
