Saving Turing Machines

 

 

VisuTuring files are written in XML format and have extension .vt. This is particularly useful if you want to open the file and make a few changes since it is a human-readable file format.

To save a file:

  1. Go to the File menu and select Save... or Save As...

  2. Choose a name and location for your file.
  3. Press Save.
  4. The file will be saved.

NB: If you want to add a file to the samples simply save it in the Samples folder that comes with VisuTuring.
