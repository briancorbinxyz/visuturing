Creating Turing Machine Transition Diagrams

 

 

VisuTuring allows you to create or generate Transition Diagrams that can be printed, saved in graphic format or used in simulations.

The initial steps for editing diagrams for a new Turing Machine are the same as with the formal methods. Simply press okay when you are satisfied with your description, states and alphabet. You can then create transitions between existing states using the diagram editor.

TIP: You can press okay as soon as the New Turing Machine Wizard opens and change its description, states and symbols at a later time if you wish.

  1. Open your Turing Machine.
  2. Go to the Edit Menu and select Transition Diagram or simply choose the Diagram tab in the Window.
  3. The Diagram Editor is displayed.

The labelled buttons function as follows:

A. Select Tool - Move states, edit and move transitions.
B. Transition Tool - Create transitions between states.
C. Recursive Transition Tool - Create transitions returning to the same state.
D. Delete Tool - Remove transitions from the Turing Machine.
E. Determinism Evaluator - Evaluate whether or not this Turing Machine is deterministic or is non-deterministic.
F. Description Button - Edit the name and description of this Turing Machine
G. States Button - Edit the states of this Turing Machine
H. Alphabet Button - Edit the symbols used in this Turing Machine
I. Generate Button - Attempt to generate or regenerate the Turing Machine based on the transition table.
J. Zoom Buttons - Zoom in and out on the diagram.

Short-Cut: Right-Clicking when in any tool will return to the Select Tool.

Creating Transitions

  1. Select the Transition Tool.
  2. Click on the first starting state for the transition.
  3. Click on the end state for the transition.

  4. A window appears where you can choose the symbol that is being read and the task to perform when it reads it.

  5. Choose the symbols from the lists and press the tick button.

 

Editing Transitions

  1. Select the Select Tool.
  2. Click on the transition you want to edit - It should turn darker and thicker and a round control point should appear.

  3. Select the control point and drag it to the desired new position.

Deleting Transitions

  1. Select the Delete Tool.
  2. Click on the transition you want to remove - a red cross appears as you hover over it.

  3. Verify that you want to remove the transition from the message box and press ok if that is the case and cancel otherwise.
  4. The transition will be removed from the diagram and Turing Machine.
