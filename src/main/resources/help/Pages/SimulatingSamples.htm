Simulating Sample Machines

 

 

VisuTuring comes with a selection of sample Turing Machines. Using these you can get a feel for how Turing Machines work, and how to use VisuTuring.

There are several ways to get started:

  1. From the startup panel choose Explore.

  2. The Samples dialog will be displayed.

If you haven't just opened VisuTuring you can choose samples from the File menu.

  1. Go to the File menu and select Open Sample...
  2. The Samples dialog will be displayed.

  3. Scroll through the list of samples and double-click on the one you want to open.
  4. You can then use either the Human Simulator or the Diagram Simulator to simulate computations on any given valid input.
