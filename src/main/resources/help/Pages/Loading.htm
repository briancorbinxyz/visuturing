Saving Turing Machines

 

 

VisuTuring files are written in XML format and have extension .vt.

There are two ways to load a file:

From the startup panel:

  1. Select Open.

From the main window:

  1. Go to the file menu and select Open...

    2.  

  2. The Open dialog will be displayed.

  3. Select your file.
  4. Press Open.
  5. If the file is a valid VisuTuring file it will be opened with no problems.
