Exporting Turing Machines to HTML

 

 

VisuTuring files are written in XML format and have extension .vt this makes conversion to any number of other file formats simple. You can create HTML representations of your Turing Machine this way:

  1. Go to the File menu and select Export To HTML File...

  2. Choose a name and location for your file.
    IMPORTANT: You should save HTML files in the same in the Web folder that comes with the main files of VisuTuring or have the images directory in the same location as the the HTML file in order for images to displayed properly.
  3. Press Save.
  4. If VisuTuring was successful it will display a message saying 'Exported Successfully'.
