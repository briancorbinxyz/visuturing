Features

 

 

VisuTuring allows you to:

 

Create
  • Specify Single Tape Turing Machines
  • Edit Single Tape Turing Machines
  • Create Turing Machine Transition Diagrams
Export & Print
  • Print Transition Diagrams
  • Save Files in Editable XML Format (.vt / .xml)
  • Export Transition Diagrams as Bitmap Files
  • Export Turing Machines to HTML Files
Simulate
  • Simulate Computations using a Transition Diagram
  • Simulate Computations using the Human Computer™
  • Simulate example Turing Machines provided with VisuTuring
