Changing the Look & Feel

 

 

VisuTuring allows you to customize the way the GUI of VisuTuring is presented.

The default Graphical User Interface is designed to Look & Feel like that on your system if it has one available for the particular Operating System you are using.

To set the local system look and feel:

  1. Go to the Configure Menu.
  2. Choose Look & Feel.
  3. Select System Look & Feel.

To set the Java look and feel:

  1. Go to the Configure Menu.
  2. Choose Look & Feel.
  3. Select Java Look & Feel.
