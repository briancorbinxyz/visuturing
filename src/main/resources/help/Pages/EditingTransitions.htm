Editing the Transition Table

 

 

Changing the transitions of a VisuTuring Turing Machine can be done by editting the transition table directly, where a transition is denoted by a tuple (current state, current symbol, next state, task):

  1. If it isn't already open, go to the Edit Menu and select Table...
  2. The Edit Transition Table Dialog will be displayed.

  3. To add a new transition, first click the Add Transition button to add an empty row at the bottom of the table. You can add as many transitions as you wish.
  4. Edit your transition by selecting the current state, current symbol - next state and task from the lists. (You must choose from all four boxes for the transition to be changed.)
  5. Remove a transition by selecting the row it corresponds to and pressing the Remove Transition button.
  6. You can have VisuTuring generate a Transition Diagram based on your transition table by pressing the Generate Transition Diagram button. (NB: This is by no means the best way to create a diagram and I recommend you edit the diagram after for best results and readability)
  7. If you are finished, press Okay.
  8. You can cancel any changes by pressing the Cancel button.
