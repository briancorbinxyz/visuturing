Editing the Description

 

 

Changing the description and name of your Turing Machine can be done from the Edit Description Panel:

  1. If it isn't already open, go to the Edit Menu and select Description...
  2. The Edit Description Dialog will be displayed.

  3. Change the name and/or description of the Turing Machine.
  4. If you are finished, press Okay.
  5. You can cancel any changes by pressing the Cancel button.
