Editing the Alphabet

 

 

You can add symbols to the input alphabet by using the Edit Alphabet Dialog.

  1. If it isnt already open, go to the Edit Menu and select Alphabet...
  2. The Edit Alphabet Dialog will be displayed.

  3. To add symbols to your alphabet type in the symbol you want to add and press the Add button.
  4. Common symbols can be added by clicking on the button corresponding to them. i.e. the Add {0, 1} button will add 0 and 1 to the input alphabet.
  5. To remove a symbol, select it and press the remove button.
  6. To remove all symbols except the left-end-marker and blank symbol, simply press the Remove All button.
  7. If you are finished, press Okay.
  8. You can Cancel any changes by pressing the Cancel button.

NB: VisuTuring will stop you from removing any symbols involved in transitions. You must remove the transitions first.
