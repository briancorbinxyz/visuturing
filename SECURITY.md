# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.1   | :white_check_mark: |
| < 2.0.1 | :x:                |

## Reporting a Vulnerability

Send a message to github@briancorbin.co.uk, though this project is not under active development at present so patches may be infrequent.
